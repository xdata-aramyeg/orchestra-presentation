The problem

One giant agent grades its own homework.

What goes wrong

  • It writes the code and reviews the code.
  • Blind to its own assumptions — no second pair of eyes.
  • One context window holds everything until it overflows.
  • "Looks done to me." Ships the bug it just wrote.

The thesis

Quality isn't a prompt trick. It's structural.

Quality =
separation of concerns
+ independent review.

Frame the whole talk: a single model marking its own work has no adversary. The fix mirrors how human teams work — split the roles, and make review independent. Meanwhile: Wave 1 (Researcher + Designer) is running.

The centerpiece

Async fan-out, sync barriers

Wave 1 · async Researcher Designer
BARRIER
Wave 2 · async Frontend Backend
▶ GATE ▶
Serial · sync QA Reviewer
LIVE 🟢

Async (the boxes): independent agents run in parallel. No waiting, no shared state — that's where the speed comes from.

Sync (the bars): the orchestrator waits at each barrier until everything is in. That's where the quality comes from.

Kitchen analogy: the line cooks (Frontend, Backend) all work at once — async. But the dish doesn't leave the pass until the head chef (QA, then Reviewer) checks it — sync. Parallelism for speed; the barrier for quality. You need both. Status: Wave 2 (Frontend ∥ Backend) is about to start / running.
Inside the Designer

How a design gets real taste

1

ui-ux-pro-max

A skill that chooses the style, palette, type pairing and motion — a real design point of view, not defaults.

2

Figma MCP

The design lives in Figma. Agents read it back as tokens — colors, spacing, type — the single source of truth.

3

21st.dev magic MCP

Generates polished components from those tokens — production-grade UI, not lorem-ipsum boxes.

Taste → tokens → components. Each step hands clean, structured input to the next.

This is why the output doesn't look like generic AI. The Designer makes deliberate choices, stores them as tokens in Figma, and the component generator builds from those tokens. Status: Wave 2 (Frontend ∥ Backend) is running now — point at the screen if it's up.
