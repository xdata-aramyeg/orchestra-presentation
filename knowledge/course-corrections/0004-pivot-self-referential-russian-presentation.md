# 0004 — Pivot: self-referential Russian presentation, high-end design

- **Date:** 2026-06-17
- **Applies to:** all roles / project framing + design
- **Trigger:** The build had become "a product that doesn't exist." The human: it doesn't
  show how Claude Code teams work, which agents we used, the structure, or our mistakes; the
  design is "dark and tasteless… AI slop… embarrassing." It must be self-referential and in
  Russian.
- **Correction:** The site is a **self-referential presentation** about *this* agent team —
  how Claude Code agent teams work, the agents (character pages), the structure, and the
  mistakes we made (this log). All visitor copy is **in Russian**. Design is **light,
  editorial, high-end** — never dark AI-slop. There is a `/goal` page. We iterate
  (build → screenshot → critique → improve) until it genuinely looks great; we do not settle.
- **Why:** A demo about agentic teams should *demonstrate the team*, honestly — including its
  failures — not market a fake product. And a presentation that looks like AI-slop undermines
  the whole point. See `docs/GOAL.md`.
