# 0002 — The Lead orchestrates; it never hand-codes

- **Date:** 2026-06-17
- **Applies to:** Lead (the main session) / all phases
- **Trigger:** The Lead started hand-writing the app's code instead of delegating to the devs.
- **Correction:** The Lead **decomposes, routes, and gates only**. All feature code is
  delegated to the Frontend and Backend devs. If the Lead is tempted to "just fix it," that's
  the signal to assign/spawn a dev instead.
- **Why:** If the Lead writes the code, there is no separation of concerns and no independent
  review — the whole point of a team is lost. The Lead's value is coordination and judgment
  about what runs in parallel vs. what waits.
