# Сценарий и раскадровка фильма (RU) — для Хроникёра

> Самореферентный фильм ~75 секунд о том, как *эта* команда агентов пришла к сборке
> сайта. На русском. Только из того, что **реально** произошло — источники:
> `knowledge/thread-and-timeline-ru.md`, `knowledge/course-corrections/`,
> `knowledge/site-content-ru.md`. Контент/раскадровка, не код.
>
> **Стек (для Хроникёра):** Remotion (композиция) + GSAP (тайминг/easing),
> Three.js / react-three-fiber (3D-сцена «барьера» и сборки), Lottie (иконки/переходы),
> встраивается в сайт через `@remotion/player`. Тон: честный, сухой, уверенный —
> ошибки часть истории, не сглаживать.
>
> **Тайминг:** 30 fps. Указаны кадры (frames) и секунды. Итог: **12 сцен, ~75 с (2250 кадров)**.

---

## Действующие аватары (6 + рассказчик-Хроникёр за кадром)

- **Маэстро** — Лид/Оркестратор (дирижёрская палочка; жесты раздают работу). Цвет/сторона: центр.
- **Либреттист** — PM (перо/страница партитуры).
- **Сценограф** — Frontend (рамка сцены/занавес).
- **Машинист** — Backend (шестерёнка/рычаг за кулисами).
- **Камертон** — QA (камертон, который звенит = вердикт).
- **Рецензент** — Reviewer (лупа/печать; не держит карандаш — только читает).
- **Человек** — заказчик (вне оркестра; его реплики «прилетают» в кадр крупным текстом).
- **Хроникёр** — Motion/Видео (камера; появляется лишь в финале — он же снимает этот фильм).

**Палитра:** светлая, редакторская, как инженерный кейс. Никакого неона, тёмного фона,
фиолетовых градиентов — это сознательно (см. course-correction 0004).

## Раскадровка (scene-by-scene)

```
film:
  fps: 30
  total_seconds: 75
  total_frames: 2250
  scenes:

  - n: 1
    label: «Бриф»
    frames: 0–150        # 0–5 c
    action: На белом листе проступает одна строка-задача; вокруг неё разворачивается
            пустая «сцена» оркестра — шесть пустых пультов.
    avatars: []
    caption: «Задача: построить сайт о том, как работает команда агентов.»
    narration: «Всё началось с одной строки.»

  - n: 2
    label: «Команда собирается»
    frames: 150–360      # 5–12 c
    action: Шесть аватаров по очереди занимают свои пульты — Маэстро в центре, остальные
            полукругом. У каждого подпись-роль и метка «Opus».
    avatars: [Маэстро, Либреттист, Сценограф, Машинист, Камертон, Рецензент]
    caption: «Шесть агентов. Шесть ролей. Все — на Opus.»
    narration: «Дирижёр, аналитик, фронтенд, бэкенд, тестировщик, ревьюер.»

  - n: 3
    label: «Партитура»
    frames: 360–540      # 12–18 c
    action: Либреттист пишет — из пера выходит «партитура»: эпик → задачи → замороженный
            контракт API. Линии тянутся к Сценографу и Машинисту.
    avatars: [Либреттист, Маэстро]
    caption: «Сначала — план: эпик, задачи, контракт.»
    narration: «Либреттист пишет, что произойдёт на сцене.»

  - n: 4
    label: «Две волны параллельно»
    frames: 540–810      # 18–27 c
    action: Сцена делится на две дорожки. Слева Сценограф собирает UI, справа Машинист —
            API/БД. Дорожки идут синхронно, не пересекаясь (разные файлы).
    avatars: [Сценограф, Машинист, Маэстро]
    caption: «Волна 2: фронтенд и бэкенд строят одновременно. Разные файлы, общий контракт.»
    narration: «Параллельно — потому что они не мешают друг другу.»

  - n: 5
    label: «Барьер держит»
    frames: 810–1020     # 27–34 c
    action: Обе дорожки упираются в светящуюся вертикальную «планку» (3D-барьер, r3f).
            Маэстро поднимает палочку — барьер держится, пока страница не отрисуется и
            API не ответит. Две галочки зажигаются → барьер опускается.
    avatars: [Маэстро, Сценограф, Машинист]
    caption: «Барьер: фича готова. Дальше — только когда и страница, и API живы.»
    narration: «Лид держит барьер. Вперёд он не забегает.»

  - n: 6
    label: «Гейт QA — камертон звенит, разработчики простаивают»
    frames: 1020–1290    # 34–43 c
    action: Камертон выходит вперёд и ударяет камертон — он звенит (волны расходятся = тест
            в реальном браузере). В этот момент Сценограф и Машинист замирают, тускнеют,
            на них значок «idle». Появляется вердикт: «PASS 9/10 → 10/10».
    avatars: [Камертон, Сценограф, Машинист, Маэстро]
    caption: «Пока QA тестирует — разработчики простаивают. Под тестом ничего не меняется.»
    narration: «Удержанный барьер и есть то, что делает вердикт настоящим.»  # 0001

  - n: 7
    label: «Ревью — только читать»
    frames: 1290–1470    # 43–49 c
    action: Рецензент проводит лупой по строкам партитуры. В руках нет карандаша — он
            подсвечивает строку, но не правит. Ставит печать «чисто».
    avatars: [Рецензент, Маэстро]
    caption: «Ревьюер читает — и только после PASS. Править не может — поэтому называет вслух.»
    narration: «Никто не проверяет собственную работу.»

  - n: 8
    label: «Ошибка: лид сам кодил»
    frames: 1470–1620    # 49–54 c
    action: Флешбэк-вставка (приглушённый кадр): Маэстро сам хватает инструмент и пишет
            код. Прилетает реплика Человека крупным текстом — экран вздрагивает.
    avatars: [Маэстро, Человек]
    caption: «„Я вижу, как ты сам пишешь код. Это ужасно. Оркеструй, а не кодь.“»
    narration: «Это была ошибка. Лид убрал руки от кода.»  # 0002

  - n: 9
    label: «Сброс»
    frames: 1620–1800    # 54–60 c
    action: Вторая реплика Человека: «несуществующий продукт… позорно, сделай лучше».
            Вся сцена — тёмный «AI-slop»-вариант — осыпается/обнуляется до чистого белого
            листа. v1 уезжает в архив (ярлык archive/v1-build).
    avatars: [Человек, Маэстро]
    caption: «„Это просто описывает несуществующий продукт. Сделай лучше.“ → Сброс в чистый лист.»
    narration: «Сброс — это решение, а не поражение.»  # 0004

  - n: 10
    label: «Переопределение команды»
    frames: 1800–1980    # 60–66 c
    action: На чистом листе заново зажигаются роли — теперь с правилами-подписями:
            «Лид не кодит», «QA только тестирует», «idle-during-QA», «Reviewer только читает».
            Светлая редакторская палитра встаёт на место тёмной.
    avatars: [Маэстро, Либреттист, Сценограф, Машинист, Камертон, Рецензент]
    caption: «Переопределили команду: 6 ролей, правила качества, светлый дизайн.»
    narration: «Из ошибок и сложились правила.»  # 0001, 0002, 0003, 0004

  - n: 11
    label: «Сайт собирает сам себя»
    frames: 1980–2160    # 66–72 c
    action: Из партитуры, дорожек и галочек складываются реальные блоки сайта — hero,
            «как работают команды», структура, ошибки, страницы агентов — экран
            «печатает» сам себя в финальный лейаут.
    avatars: [Сценограф, Машинист, Камертон, Рецензент, Либреттист, Маэстро]
    caption: «Эту страницу спланировала, построила, протестировала и проверила команда из шести агентов.»
    narration: «Сайт, который построил сам себя.»

  - n: 12
    label: «Этот фильм — тоже команда»
    frames: 2160–2250    # 72–75 c
    action: В кадр въезжает седьмой аватар — Хроникёр с камерой — и наводит её на уже
            собранный сайт. Камера складывается в логотип Orchestra.
    avatars: [Хроникёр]
    caption: «Этот фильм тоже сделан командой.»
    narration: «Снято Хроникёром. На Opus. С настоящими ошибками по пути.»
```

## Заметки для Хроникёра (не выводить в фильм)

- **Арка:** бриф → команда собирается → две волны параллельно → барьер → гейт QA (idle)
  → ревью → ошибки и сброс → переопределение → сайт собирает сам себя → «фильм тоже команда».
- **3D (r3f):** барьер (сцена 5) и сборка сайта (сцена 11) — главные кандидаты на 3D;
  остальное можно держать 2D (GSAP + Lottie), чтобы не перегружать.
- **Реплики Человека** (сцены 8–9) — это удар: крупный текст, дрожь кадра, смена палитры.
  Не смягчать формулировки — резкость и есть смысл.
- **Палитра-переход:** до сброса допустима тёмная «AI-slop» вставка как *показ ошибки*;
  после сброса (сцена 10) — окончательно светлая редакторская. Это сознательный контраст.
- **Кросс-ссылки на course-corrections** проставлены в `narration` (# 0001–0004).
- **Continuously improved:** сценарий — живой; при правках сюжета обновлять и этот файл.
- **Длительность:** держим 60–90 с. Текущая раскладка — 75 с / 2250 кадров при 30 fps;
  при необходимости ужать сцены 3 и 7 на ~30 кадров каждую без потери арки.
